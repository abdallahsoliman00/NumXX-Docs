zeros()

Creates an array of a given shape, filled with zeros.

Syntax

template <typename T = double, typename = std::enable_if_t<std::is_arithmetic_v<T>>>
NArray<T> zeros(const Shape& shape);
template <typename T = double, typename = std::enable_if_t<std::is_arithmetic_v<T>>>
NArray<T> zeros(const size_t& size);

Parameters

T : (defaults to double)
The data type to be stored.
shape : Shape
Dimensions of the shape to create.
size : size_t
The size of the array (creates a 1D array).

Returns

An NArray object filled with zeros.

Example

#include <NumXX.hpp>

int main() {
    auto mat = numxx::zeros<float>({2,3});
    auto arr = numxx::zeros<float>(5);

    std::cout << mat << std::endl;
    // >> [[0 0 0]
    //     [0 0 0]]
    std::cout << arr << std::endl;
    // >> [0 0 0 0 0]
}
