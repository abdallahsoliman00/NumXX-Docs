zeros()

Creates an array filled with zeros.

Syntax

Array zeros(const Shape& shape, DataType dtype = FLOAT64)

Parameters

Parameter Type Description
shape Shape The dimensions of the array to create
dtype DataType The data type of the array elements (default: FLOAT64)

Returns

An Array object filled with zeros of the specified shape and data type.

Example

#include <numxx/numxx.hpp>

int main() {
    auto arr = numxx::zeros({3, 4});
    arr.print();
    return 0;
}

Parameters

Parameter Type Description

arange()

Creates an array with evenly spaced values within a given interval.

Syntax

Array arange(double start, double stop, double step = 1.0)

Parameters

Parameter Type Description
start double Start of interval (inclusive)
stop double End of interval (exclusive)
step double Spacing between values (default: 1.0)

Example

auto arr = numxx::arange(0, 10, 2);  // [0, 2, 4, 6, 8]

linspace()

Creates an array with evenly spaced values over a specified interval.

Syntax

Array linspace(double start, double stop, size_t num = 50)

Parameters

Parameter Type Description
start double Starting value of the sequence
stop double End value of the sequence
num size_t Number of samples to generate (default: 50)

Example

auto arr = numxx::linspace(0, 1, 5);  // [0.0, 0.25, 0.5, 0.75, 1.0]
